# Clash of Prompt - AI Battle Arena

A turn-based battle RPG where your creativity is your weapon. Write prompts to attack enemies, and an AI Game Master narrates the battle, scores your creativity, and determines damage. Built on its own Initia Move Minitia rollup (`clashofprompt-1`) with on-chain scoring, CLASH token rewards, and a live leaderboard.

**Live:** [clash-of-prompt.vercel.app](https://clash-of-prompt.vercel.app) | **Backend + Chain:** [clash-of-prompt-full.fly.dev](https://clash-of-prompt-full.fly.dev)

## Architecture

Split into two services, with the backend running an embedded Initia node:

```
clash-of-prompt/
├── src/                    Frontend (Vercel)
│   ├── app/api/            Thin proxies to backend
│   ├── components/         React UI components
│   └── lib/i18n.ts         Internationalization (EN/ID)
│
├── backend/                Backend + Chain Node (Fly.io)
│   ├── server.ts           HTTP server
│   ├── lib/                Game engine, Claude CLI, Gemini, chain module
│   ├── Dockerfile
│   └── fly.toml
│
├── deploy/                 Move smart contracts
│   └── game_arena/         CLASH token, leaderboard, scores, PvP
│
└── .initia/
    └── submission.json     Hackathon submission metadata
```

- **Frontend**: Next.js 16 + React 19 + Tailwind CSS 4. Deployed on Vercel at [clash-of-prompt.vercel.app](https://clash-of-prompt.vercel.app).
- **Backend + Chain Node**: Node.js HTTP server with game logic, Claude CLI (OAuth) for AI game master, Gemini API for battle images, and an embedded Initia node for on-chain operations. Deployed on Fly.io at [clash-of-prompt-full.fly.dev](https://clash-of-prompt-full.fly.dev).

## Features

- Turn-based RPG combat with AI-narrated battles
- Creativity scoring system (1-10) that scales damage
- Status effects: poison, stun, shield, heal, burn
- 3 enemies with unique personalities and weaknesses
- Comic-style battle images generated by Gemini
- Bilingual support (English / Bahasa Indonesia)

## On-Chain Features

All game state lives on the `clashofprompt-1` Initia Move Minitia rollup.

- **CLASH Token** -- Fungible asset rewarded for battle victories. Reward scales with creativity score.
- **On-Chain Leaderboard** -- Scores recorded in the `game_arena` Move module. Queried directly from chain state.
- **PvP Wagers** -- CLASH tokens locked in smart contract. Winner takes all.

## Local Development

### Prerequisites

- Node.js 20+
- Docker (OrbStack recommended on macOS)
- Claude CLI authenticated (`claude auth login`)
- Gemini API key (optional, for battle images)

### Run locally

```bash
# Terminal 1: Backend (Docker)
cd backend
docker build -t clash-of-prompt-backend .
docker run -p 8080:8080 \
  -e CLAUDE_CODE_OAUTH_TOKEN=your-token \
  -e GEMINI_API_KEY=your-key \
  clash-of-prompt-backend

# Terminal 2: Frontend
npm install
npm run dev
```

Frontend runs at `http://localhost:3000`, backend at `http://localhost:8080`.

### Run backend without Docker

```bash
cd backend
npm install
npm run dev
```

Requires Claude CLI installed and authenticated locally.

## Deployment

### Backend + Chain Node (Fly.io)

```bash
cd backend
fly launch
fly secrets set CLAUDE_CODE_OAUTH_TOKEN=sk-ant-oat01-xxx
fly secrets set GEMINI_API_KEY=xxx
fly secrets set ALLOWED_ORIGINS=https://clash-of-prompt.vercel.app
fly deploy
```

The Fly.io deployment includes the embedded Initia node. The backend handles game logic, AI calls, and all chain interactions (score recording, CLASH rewards, leaderboard queries).

### Frontend (Vercel)

1. Push repo to GitHub
2. Import in Vercel
3. Set env var: `BACKEND_URL=https://clash-of-prompt-full.fly.dev`
4. Deploy
